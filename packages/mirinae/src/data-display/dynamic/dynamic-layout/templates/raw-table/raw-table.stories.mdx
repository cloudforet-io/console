## Options Example

### Options: Disable Search

```js
{
    disable_search: true
}
```

<Canvas>
    <Story name="Disable Search">

            components: { PDynamicLayout },

### Options: Headers
"headers" option is used to specify the order of the table headers.
```js
{
    headers: ['information', 'age', 'name', 'job']
}
```
